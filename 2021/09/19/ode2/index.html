《常微分方程》（方道元等）课后习题解答

Posted on 2021-09-19 Edited on 2021-09-21 In exercise

第二章第一节

1.

First order linear ode has the form $\frac{\mathrm{d}y}{\mathrm{d}x}=p(x)y+q(x)$.

Its general solution gives by $y=\mathrm{e}^{\int p(x)\,\mathrm{d}x}(C+\int q(x)\mathrm{e}^{-\int p(x)\,\mathrm{d}x}\,\mathrm{d}x)$.

The special solution with initial value $y(x_0)=y_0$ gives by $y=\mathrm{e}^{\int_{x_0}^x p(s)\,\mathrm{d}s}(y_0+\int_{x_0}^x q(s)\mathrm{e}^{-\int_{x_0}^s p(t)\,\mathrm{d}t}\,\mathrm{d}s)$.

(1)

General solution.

$y=\mathrm{e}^{x}(C+\int \sin x\cdot\mathrm{e}^{-x}\,\mathrm{d}x)=C\mathrm{e}^{x}-\frac{1}{2}(\sin x+\cos x)$

(2)

Special solution.

$y=\mathrm{e}^{\int_{x_0}^x p(s)\,\mathrm{d}s}(y_0+\int_{x_0}^x q(s)\mathrm{e}^{-\int_{x_0}^s p(t)\,\mathrm{d}t}\,\mathrm{d}s)=|\frac{1+x}{1-x}|^{\frac{1}{2}}(1+\int_0^x(1+s)|\frac{1-s}{1+s}|^{\frac{1}{2}}\,\mathrm{d}s)$

Notice that $|x|\le1$ and $|x|>1$ need to be discussed separately.

$|x|\le1$.

$y=(\frac{1+x}{1-x})^{\frac{1}{2}}(1+\int_0^x\sqrt{1-s^2}\,\mathrm{d}s)=y=(\frac{1+x}{1-x})^{\frac{1}{2}}(1+\frac{\arcsin x+x\sqrt{1-x^2}}{2})$.

$x>1$.

$y=(\frac{x+1}{x-1})^{\frac{1}{2}}(1+\frac{\pi}{4}+\int_1^x\sqrt{s^2-1}\,\mathrm{d}s)=(\frac{x+1}{x-1})^{\frac{1}{2}}(1+\frac{\pi}{4}+\frac{1}{2}x\sqrt{x^2-1}-\frac{1}{4}\ln\frac{x+\sqrt{x^2-1}}{x-\sqrt{x^2-1}})$.

$x<-1$.

$y=(\frac{x+1}{x-1})^{\frac{1}{2}}(1-\frac{\pi}{4}+\int_{-1}^x\sqrt{s^2-1}\,\mathrm{d}s)=(\frac{x+1}{x-1})^{\frac{1}{2}}(1-\frac{\pi}{4}+\frac{1}{2}x\sqrt{x^2-1}+\frac{1}{4}\ln\frac{-x+\sqrt{x^2-1}}{-x-\sqrt{x^2-1}})$.

(3)

General solution.

$y=\mathrm{e}^{-x^2}(C+x^2)$

(4)

Special solution.

$y=\frac{1}{|\sin x|}(1+\int_{\frac{\pi}{2}}^x \theta\cdot\frac{|\sin\theta|}{\sin\theta}\,\mathrm{d}\theta)$

Suppose $|x|=k\pi+r$, in which $k$ is an integer such that $k\pi\le|x|<(k+1)\pi$.

Then $y=\frac{1}{\sin x}[(-1)^k\frac{|x|}{x}+\frac{1}{2}r^2+kr\pi-(\frac{k}{2}+\frac{1}{8})\pi^2]$.

(5)

Special solution.

$y=\frac{\sin x-x\cos x}{x^2}$

(6)

General solution.

$y=\begin{cases}(1+x)^n(C+e^x)&n\text{ is even}\\C|1+x|^n+(1+x)^n(\mathrm{e}^x-\mathrm{e}^{-1})&n\text{ is odd}\end{cases}$

2.

Calculate the derivatives on both sides

$\begin{cases}f’=\mathrm{e}^x+f\\f(0)=1\end{cases}$

Solve this ode, we get

$f=\mathrm{e}^x(C+x)$.

3.

Calculate the derivatives on both sides

$\begin{cases}f’=2x+\int_0^x2xf’(t)\,\mathrm{d}t \\f(0)=1\end{cases}$

i. e. $\frac{f’}{2x}=1+\int_0^xf’(t)\,\mathrm{d}t $

The right side of the equation above is obviously differentiable, which means the left side is also differentiable. Calculate the derivatives on both sides again,

$\begin{cases}\frac{2xf’’-2f’}{4x^2}=f’(x)\\f’(0)=0\end{cases}$

Solve this ode, we get

$f’=Cx\mathrm{e}^{x^2}$.

So $f=C(\mathrm{e}^{x^2}-1)+1$.

4.

Consider the function $Y(x)=y(x)\mathrm{e}^{\int_0^xa(t)\,\mathrm{d}t}$. Given the conditions, we know that $Y’(x)\le0$ whenever $x\ge0$.

So we have $Y(x)\le Y(0)$ for all $x\ge0$, which immediately induces what we are asked to prove.

5.

This ode has the general solution of $y=\mathrm{e}^{x}(C+\int_0^x f(s)\mathrm{e}^{-s}\,\mathrm{d}s)$

When $x\to+\infty$, if $y$ is bounded as required, we know that there must be $\lim_{x\to+\infty}(C+\int_0^x f(s)\mathrm{e}^{-s}\,\mathrm{d}s)=0$, i. e. $\int_0^{+\infty} f(s)\mathrm{e}^{-s}\,\mathrm{d}s=-C$, which also indicates that the bounded solution must be unique if exists.

From the analysis above, we firstly need to prove that this improper integral is convergent.

Suppose $|f|0$, take $A=\ln\frac{M}{\varepsilon}$, we have $|\int_A^{+\infty} f(s)\mathrm{e}^{-s}\,\mathrm{d}s|<M\int_A^{+\infty}\mathrm{e}^{-s}\,\mathrm{d}s=M\mathrm{e}^{-A}=\varepsilon$. So the integral is convergent due to Cauchy’s test.

Now we choose $C=-\int_0^{+\infty} f(s)\mathrm{e}^{-s}$. We are going to prove that in this case, $y$ is indeed bounded.

$|y|=|\mathrm{e}^{x}\int_{+\infty}^x f(s)\mathrm{e}^{-s}\,\mathrm{d}s|\le\mathrm{e}^{x}M \mathrm{e}^{-x}=M$. Q. E. D.

If $f(x+\omega)=f(x)$ for all $x$, then we notice that $\int_x^{+\infty} f(s)\mathrm{e}^{-s}\,\mathrm{d}s=\int_x^{x+\omega} f(s)\mathrm{e}^{-s}\,\mathrm{d}s\sum_{n=0}^{+\infty}\mathrm{e}^{-n\omega}=\frac{1}{1-\mathrm{e}^{-\omega}}\int_x^{x+\omega} f(s)\mathrm{e}^{-s}\,\mathrm{d}s$.

Since $\int_x^{x+\omega} f(s)\mathrm{e}^{-s}\,\mathrm{d}s=\mathrm{e}^{-\omega}\int_{x+\omega} ^{x+2\omega} f(s)\mathrm{e}^{-s}\,\mathrm{d}s$, it is easy to see $y(x+\omega)=y(x)$.

6.

(1)

In the case $q(x)\equiv0$, it has general solution of $y=C\mathrm{e}^{\int_0^x p(s)\,\mathrm{d}s}$.

If $y(x+\omega)=y(x)$, we can easily see that $\mathrm{e}^{\int_{x}^{x+\omega} p(s)\,\mathrm{d}s}=\mathrm{e}^{\int_{0}^{\omega} p(s)\,\mathrm{d}s}=0$.

(2)

General solution $y=\mathrm{e}^{\int_{0}^x p(s)\,\mathrm{d}s}(C+\int_{0}^x q(s)\mathrm{e}^{-\int_{0}^s p(t)\,\mathrm{d}t}\,\mathrm{d}s)$.

If $y(x+\omega)=y(x)$, we have $C=\frac{1}{\mathrm{e}^{\omega\bar{p}}-1}(\int_{0}^x q(s)\mathrm{e}^{-\int_{0}^s p(t)\,\mathrm{d}t}\,\mathrm{d}s-\mathrm{e}^{\omega\bar{p}}\int_{0}^{x+\omega} q(s)\mathrm{e}^{-\int_{0}^s p(t)\,\mathrm{d}t}\,\mathrm{d}s)$ holds for all $x$. Also, this fact indicates that there can possibly be an unique periodic solution if $\bar{p}\neq0$.

Now we need to prove that the right side is irrelavant to $x$.

It is clear that the right side is differentiable to $x$. Calculate its derivatives, we have

$(\text{The Right Side})’=q(x)\mathrm{e}^{-\int_{0}^x p(t)\,\mathrm{d}t}-\mathrm{e}^{\omega\bar{p}} q(x+\omega)\mathrm{e}^{-\int_{0}^{x+\omega} p(t)\,\mathrm{d}t}\equiv0$ .

Q. E. D.
